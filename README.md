**FreeLancer** is a premium, all-in-one ecosystem designed for the modern student developer. It combines a secure freelance marketplace, direct mentorship access, and an integrated AI companion into a single, high-fidelity experience.

---

## 🛰️ Core Modules

### 💼 **Job_Buffer** (Freelance Marketplace)
*The marketplace node where students find real-world work.*
- **Vetted Projects**: Only student-ready technical and creative gigs.
- **Smart Escrow**: Secure payment handling via the Stripe protocol.
- **Verified Status**: Build your reputation through successful project completions.

### 🧠 **Mentor_Nodes** (Expert Access)
*Direct 1-on-1 uplink with industry veterans.*
- **Direct Booking**: Schedule sessions for code reviews or career roadmap audits.
- **Expert Directory**: Browse mentors from top companies and institutes.
- **Real-time Chat**: Secure communication channels for project guidance.

### 🤖 **AI_Core** (Intelligent Assistant)
*Your technical companion powered by Google Gemini 2.0 Flash.*
- **Instant Debugging**: Solve complex code hurdles in seconds.
- **Contextual Help**: Integrated across the platform to help you build faster.
- **Low Latency**: High-speed responses for a seamless building flow.

## 🛠️ Technical Architecture

FreeLancer is built on a "Cyber-Dark" stack optimized for speed and visual excellence:

- **Core**: [Next.js 15](https://nextjs.org/) (App Router) & [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [Framer Motion](https://www.framer.com/motion/) (Smooth glassmorphic transitions)
- **Identity**: [Clerk Auth](https://clerk.com/) (Secure user sessions)
- **Database**: [MongoDB](https://www.mongodb.com/) (Scalable document storage)
- **Finance**: [Stripe](https://stripe.com/) (Automated escrow and payment flows)

## 📦 Quick Start Node

### 1. Initial Setup
```bash
git clone https://github.com/princeyuviii/FreeLancer.com.git
cd FreeLancer.com
npm install
```

### 2. Environment Configuration
Create a `.env.local` file and populate it with your API keys:
```env
MONGODB_URI=your_mongodb_uri
GEMINI_API_KEY=your_gemini_key
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_pub_key
CLERK_SECRET_KEY=your_clerk_secret_key
STRIPE_SECRET_KEY=your_stripe_secret_key
RESEND_API_KEY=your_resend_key
```

### 3. Deploy Local Buffer
```bash
npm run dev
```
